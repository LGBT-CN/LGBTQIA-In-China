> People were born free and equal. 人们生来是而且始终是自由平等的

本仓库意图在 LGBTQIA 程序员群体中唤起对自由平等的追求，哪怕我们都深知这不会激起浪花。

你可以访问[这个网站](https://cnlgbt.org)查看更多信息。
